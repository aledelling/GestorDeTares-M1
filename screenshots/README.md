# Carpeta para capturas de pantalla

Aquí debes agregar dos capturas de tu aplicación:

1. **lista_tareas.png** - Captura de la pantalla principal mostrando la lista de tareas
2. **formulario_tarea.png** - Captura del formulario de creación/edición de tareas

## Cómo tomar las capturas:

### En Emulador:
- Usa el botón de cámara en la barra lateral del emulador
- O presiona Ctrl+S (Windows/Linux) o Cmd+S (Mac)

### En Dispositivo Físico:
- Generalmente: Botón Power + Volumen Abajo simultáneamente

Luego guarda las imágenes en esta carpeta con los nombres indicados.
